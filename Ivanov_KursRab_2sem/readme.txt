	Задание курсовой работы (вариант 7):

Требуется написать программу, на вход которой подаются две таблицы хрянящиеся в csv файлах. 
Первая таблица содержит информацию о ФИО студента и его Github аккаунте, email'е и номере группы. 
Вторая таблица содержит фамилию, имя, количество баллов за экзамен. 
Программа должна находить количество студентов, которые получили максимальный балл и 
выводить результат на консоль.

Программа получает параметры из входного потока. Параметры:
input_file_1 - csv файл
input_file_2 - csv файл

В случае, если программа получила некорректные параметры, то:
не создается выходного в файла
выводится сообщение об ошибке “Fail with <имя параметра>”.


Краткое описание алгоритма работы программы:

Программа получает на вход два cvs-файла, проверяет корректность файлов:
	- существование обоих файлов
	- корректность строк, содержащихся в файле
		* одновременно с проверкой корректности второго файла сохраняет ФИ и балл студента
в массив, проверяя встречался ли этот студент ранее
	- равенство кол-ва строк в файлах

После проверки корректности файлов программа просматривает input_file_1 (в цикле),
ища в нём студентов из массива студентов, при существовании и единственности данного студента 
возвращает его балл за экзамен. Далее балл студента сравнивается с максимальным:
	- если балл == макс. балл - увеличивает счётчик
	- если балл > макс. балл - сбрасывает счётчик и записывает значение балла в макс. балл
	- если балл < макс. балл - не делает ничего

После удачного нахождения всех студентов и подсчёта кол-ва с максимальным баллом, 
выводит это кол-во на консоль.


Помимо стандартных ситуаций (корректных и правильных файлов) предусмотрены ситуации:
	+ отсутствие одного/обоих файлов
		- выводится "input_file_1(2).csv doesn't exist"
	+ разное кол-во студентов в файлах
		- выводится "Failed with <Different Numbers of Students>"
	+ отсутствие: в 1ом файле Фамилия/Имя/Гит/Мэйл/Группа, во 2ом - Фамилия/Имя/Балл
		- в зависимости от отсутствующего параметра выводится сообщение
			Failed with <Surname. File №1>
			Failed with <Name. File №1>
			Failed with <Score>
			Failed with <Git>
			Failed with <Mail>
			Failed with <Group>
			Failed with <Surname. File №2>
			Failed with <Name. File №2>
		- также возможно отсутствие отчества в 1ом файле (но некоторые студенты его не имеют)
	+ больше одного студента имеют одно имя-фамилию в одном из файлов
		- выводится "Failed with <One name for a few students. File №1(2)"
	+ отсутствие студента из 2ого файла в 1ом (в случае одинакового кол-ва студентов в файлах)
		- выводится "Failed with <Student isn't found>"

Во вложенных файлах input_file_1.csv и input_file_2.csv содержатся входные данные на ~300 студентов.
(MS Excel, к сожалению, делает разделитель ";" вместо ",", но программа рассчитана на оба разделителя)
Т.к. в первом файле проверяется наличие Mail и Git, эти столбцы заполнены посторонней информацией,
чтобы они не были пусты (заполнять 600 ячеек Mail и Git осмысленными строками затруднительно)
